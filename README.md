# Dijkstra's Algorithm
This program visualizes [Dijkstra's Algorithm](https://en.wikipedia.org/wiki/Dijkstra's_algorithm) for finding the shortest path between two points on a weighted graph.

## Usage
```bash
# Install Dependencies
pip install -r requirements.txt

# Run Program
python3 main.py
```

Click any two points and the program will run Dijkstra's Algorithm to find the shortest path between them.  
Click Restart Algorithm to selected new points, Click Regenerate Graph (or press R) to generate a new Graph.
